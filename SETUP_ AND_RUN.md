# 🚀 Setup and Run Fresh Web Lite

## Step 1: Install Required Programs

Before you start, download and install these on your computer:

### 1. Node.js
- Go to: https://nodejs.org/
- Download the **LTS version**
- Install it (just click through)
- Open Command Prompt and type: `node --version` (should show a version number)

### 2. Git
- Go to: https://git-scm.com/
- Download and install
- Open Command Prompt and type: `git --version` (should show a version number)
